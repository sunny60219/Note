## ER Model表
### 面試

| 名稱 | 作用 | 
| --- | --- |
| Interview(面試) | 儲存一次面試的所有資訊，包含面試者、被面試者、時間、地點、類型等等 |
| Resume(履歷資料) | 儲存面試者填寫的所有基本資訊 + 履歷上檢核項目，包含姓名、生日、最後上班日、可報到日期等等 |
| ReferenceCheck(資歷查核) | 儲存HR在面試中紀錄填入的資料，包含細心度、離職原因、目標達成率 |
| Euacation(教育程度) | 儲存**面試者面試前**填寫的學歷資訊，如大學的開始日期、結束日期等等，同一位面試者可有多筆(高中、大學) |
| WorkExperence(工作經歷) | 儲存**面試者面試前**填寫的工作經歷資訊，如離職原因、主管姓名等等 |
| Language(語文能力) | 儲存**面試者面試前**填寫的語文能力資訊，如英文的聽說讀寫等等 |
| SuperVisor(前公司主管) | 儲存**面試者面試前**填寫的前主管資訊，如姓名、電話等等 |
| Family(家庭狀況) | 儲存**面試者面試前**填寫的家庭成員資訊，如關係、姓名、職業等等 |
